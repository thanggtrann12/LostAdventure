## Basic control

| Control     | Action              |
|:------------|:--------------------|
| **Left Mouse**  | **Basic Attack**        |
| **Right Mouse** | **Move**                |
| **Tab**         | **Toggle Inventory**    |
| **M**           | **Toggle Map**          |
| **K**           | **Toggle Skill Tree**   |
| **Q**           | **Skill 1**             |
| **W**           | **Skill 2**             |
| **E**           | **Skill 3**             |
| **R**           | **Skill 4**             |
| **Shift**       | **Dash**                |
| **A**           | **Show Attack Range**   |
